# 🎨 Система распознавания рукописных цифр

Приложение для распознавания рукописных цифр (0-9) с использованием нейронной сети на основе датасета MNIST.

## 📋 Требования

- Python 3.12+
- TensorFlow 2.13+
- Pillow 10.0+
- NumPy 1.24+

## 🚀 Запуск

```bash
# Установка зависимостей
uv sync

# Запуск приложения
python main.py
```

## 💻 Использование

1. **Рисование**: Используйте мышь для рисования цифры на белом холсте
2. **Распознавание**: Нажмите кнопку "Распознать" для определения цифры
3. **Результат**: Программа покажет распознанную цифру и уверенность предсказания
4. **Очистка**: Нажмите "Очистить" для стирания холста и повторной попытки

## 🧠 Как это работает

- **Модель**: Обычная нейронная сеть (Neural Network) с одним скрытым слоем
- **Обучение**: Модель обучена на датасете MNIST (70,000 рукописных цифр)
- **Интерфейс**: Простой и интуитивный интерфейс на основе tkinter
- **Обработка**: Рисунок масштабируется до 28x28 пикселей (стандарт MNIST)

## 📊 Архитектура модели

```
Input (28x28)
    ↓
Flatten
    ↓
Dense(128, relu)
    ↓
Dropout(0.2)
    ↓
Dense(10, softmax)  [Цифры 0-9]
```

- ✅ Быстрое распознавание
